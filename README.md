## Описание функционала

Приложение представляет собой **галерею изображений котов** с возможностью поиска по породам. Основные функции:

- **Поле ввода для поиска** — позволяет искать изображения по породам котов.
- **Грид картинок** — все изображения отображаются аккуратно в ряд с одинаковым размером.
- **Ленивая загрузка (infinite scroll / кнопка «Загрузить ещё»)** — новые картинки подгружаются по кнопке.
- **Lightbox** — клик по изображению открывает его в большом размере.
- **Фильтры / сортировка** — кнопки «Старые / Новые» позволяют менять порядок отображения изображений.
- **История поиска** — сохраняются последние 10 запросов, с возможностью:
  - нажать на элемент истории для повторного поиска
  - удалить отдельный элемент
  - очистить всю историю
- **Сообщение при неправильном вводе** — если порода не найдена, выводится текст «Нет такой породы».

## Используемое API

- [TheCatAPI](https://thecatapi.com/) — для загрузки изображений и списка пород котов.

---

## Примеры запросов

**Получение случайных изображений котов:**

GET https://api.thecatapi.com/v1/images/search?limit=12&page=0&order=DESC

## Получение изображений определённой породы:
GET https://api.thecatapi.com/v1/images/search?limit=12&page=0&order=DESC&breed_ids=beng

## Получение списка пород котов:
GET https://api.thecatapi.com/v1/breeds

Какие ошибки возникли при интеграции

Кнопки «Старые / Новые» не работали — при пустом search или пустом breedId API не сортировал картинки.

История поиска не отображалась сразу — useEffect срабатывал только после перезагрузки страницы.

При неправильном вводе — раньше галерея просто оставалась пустой, без сообщения.

Как вы решили возникшие ошибки

Кнопки сортировки — добавлен сброс страницы и очистка массива изображений при смене order, order всегда передается в API.

История поиска — хранится в состоянии App и обновляется динамически, сразу отображается при вводе.

Сообщение при неверной породе — добавлено состояние notFound в Gallery.jsx, выводит текст вместо пустой галереи.

Результат:
Рабочее React-приложение с полноценной галереей изображений котов, поиском по породам, фильтрами, сортировкой, Lightbox, историей и кнопкой «Загрузить ещё». Готово к запуску и сдаче лабораторной работы.
